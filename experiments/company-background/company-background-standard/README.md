# 公司背景（Company Background）

> 负责人：@Dongjianye | 领域：公司级

## 这个 Skill 解决什么问题

团队成员在写产品文案、对外沟通、给新同事介绍公司时，需要准确一致的公司定位和产品描述。如果每个人凭印象写，容易出现数据过时、定位偏差、用语不一致的问题。这个 skill 提供草料的权威背景资料，确保 AI 在涉及公司信息时给出准确、统一的内容。

## 什么时候会触发

当你让 AI 帮你做以下事情时会自动生效：
- 写产品介绍、公司简介、合作方案
- 向客户或合作伙伴介绍草料
- 理解某个产品决策背后的业务逻辑
- 新入职了解公司业务全貌

## 设计考量

内容按"身份 → 愿景 → 文化 → 产品 → 用户价值 → 行业"的层次组织，从抽象到具体。数据（1600 万用户、43 亿二维码）需要定期更新以保持准确。详细的客户案例和产品细节放在 `references/` 子目录，避免主文件过长。
