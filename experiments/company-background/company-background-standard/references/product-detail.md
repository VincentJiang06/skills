# 草料二维码 — 详细产品与背景参考

## 公司历程

- **2012 年**：草料二维码上线，提供免费二维码生成服务
- **2012–2016 年**：普及活码（动态二维码）概念，让用户理解"二维码内容可以改"
- **2016 年至今**：从二维码工具进化为轻量业务系统平台，服务企业数字化管理
- **当前规模**：1600 万+ 用户，43 亿+ 二维码生成量

## 公司理念

### 核心信念

"技术不应该成为门槛。工具应该是人人都用得起、用得会的，而不是只有有钱的少数人才能享用。"

### 经营哲学

- 没有销售团队，不投放付费广告，靠产品口碑自然增长
- 通过低成本运营补贴免费版，让基础功能对所有人开放
- 14 年持续运营积累的信任是核心壁垒

### 做事标准

- 产品设计以一线操作人员为第一视角
- 功能迭代优先解决高频痛点，不做花哨但无用的东西
- 安全合规是底线（等保三级、ISO 27001、区块链存证）

## 产品体系详解

### 第一层：基础二维码生成（免费）

- 无限量生成，支持 QR/汉信码/PDF417/DataMatrix
- 纠错等级 7%–30%
- 输出格式：PNG / PDF / SVG / EPS
- 本地编码工具（敏感数据不过服务器）
- 浏览器插件快捷生码

### 第二层：动态内容管理（活码）

- 富文本编辑器：文字、图片、视频、文件、音频
- 内容修改后二维码图片不变，无需重新印刷
- 20+ 展示模板
- 链接追踪与扫码统计
- 免费版保留 180 天数据导出

### 第三层：表单与数据采集

- 拖拽式表单搭建
- AI 组件：身份证识别、文字识别等
- 签名字段、拍照水印（时间/地点/人员）
- 实时消息通知（微信/企微/钉钉）
- 数据审计追踪、区块链存证

### 第四层：企业协作功能（付费）

- 多人协作，角色权限管理
- 批量模板生码（Excel 导入）
- A4 标签设计与打印（150+ 模板）
- 数据 API 和 Webhook 集成
- 高清媒体（无水印）
- 自定义品牌与域名

### 第五层：行业场景方案

- 设备巡检模板
- 资产管理系统
- 安全检查清单
- 人员签到/签退
- 物料取样追踪
- 设备故障记录
- 预防性维护排程

## 目标市场与客户案例

### 制造业

**痛点**：物料追踪靠纸质记录，设备维护无体系化管理
**方案**：一物一码 + 表单采集 + 状态管理
**案例**：某设计工作室通过草料管理物料样品，管理效率提升 10 倍

### 建筑工程

**痛点**：工地分散，安全管理靠人盯人，项目文档纸质化
**方案**：巡检二维码 + 计划管理 + 数据面板
**案例**：杭州地铁 3 号线项目数字化管理

### 医疗卫生

**痛点**：医疗器械管理复杂，培训记录难追溯
**方案**：设备档案码 + 巡检表单 + 培训签到
**案例**：盘锦医院 ICU 设备、耗材、人员培训零纸化管理

### 能源矿业

**痛点**：设备运行记录不连续，故障归档困难
**方案**：设备运行日志 + 故障记录 + 预防性维护

### 物业管理

**痛点**：资产盘点耗时，设施巡检缺乏标准化
**方案**：资产二维码 + 定期巡检计划 + 状态追踪

## 核心术语速查

| 术语 | 定义 |
|------|------|
| 活码 | 动态二维码，内容可修改，图片不变（草料核心产品） |
| 静态码 | 内容直接编码在图片中，不可修改 |
| 文本码 | 将字符串转为二维码（序列号、条码替代） |
| 动态数据 | 与二维码关联的所有扫码记录和表单提交 |
| 批量生码 | 基于模板批量生成主题码 |
| 状态管理 | 追踪对象状态（正常/异常/已维护） |
| 动态档案 | 对象的完整生命周期记录 |
| 万物履历 | 公司愿景：为每个实体建立数字档案 |
| 分区 | 二维码的组织分类方式 |
| 计划管理 | 定期任务（巡检/保养）的排程与执行追踪 |
| 高级成员 | 拥有扩展权限的团队用户 |
