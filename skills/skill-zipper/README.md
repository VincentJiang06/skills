# skill-zipper

为 token 效率、可靠性与触发准确度重构现有 skill：拆分到 rules/、精简、去模糊、修复触发。
Restructures an existing skill for token efficiency, reliability, and trigger accuracy — split into rules/, compress, de-vague, fix triggering.

- **触发 Triggers** — “这个 SKILL.md 太长了，帮我拆到 rules/” · “它总是不触发，看看 description” · `/skill-zipper`
- **用法 Use** — 指向一个现有 skill 目录。Point at an existing skill folder.
- **不适用 Not for** — 从零创建 skill（→ skill-creator）、衡量多次运行的输出质量、非 skill 的写作 / 编辑。Creating from scratch, measuring output quality, or non-skill writing.
