# skill-guidance

评估一个 skill 或 skill 仓库，打分、定范围、找缺口，并产出交给 engineer 的 handoff spec（流水线 stage 1）。
Audits a skill or skill repo — scores, scopes, finds gaps — and emits the handoff spec for the engineer (pipeline stage 1).

- **触发 Triggers** — “这个 skill 够工业级吗 / 缺什么” · “audit this skill before build” · `/skill-guidance`
- **用法 Use** — 指向一个 SKILL.md 或一个 skill 目录。Point at a SKILL.md or a skill folder.
- **不适用 Not for** — 实现（→ engineer）、token 重构（→ zipper）、空白脚手架（→ skill-creator）。Implementation, token restructuring, or blank scaffolding.
