# vince-fact-check

快速、有出处的事实核查：并行检索、尽早收敛，限时给出 BLUF 结论（简单 ≤2 分钟，复杂 ≤5 分钟）。
Fast, citation-backed fact-check — parallel search, early-exit, hard time budget (≤2 min simple, ≤5 min complex).

- **触发 Triggers** — “fact-check this” · “查一下 X / X 是真的吗” · `/vince-fact-check`
- **用法 Use** — 给一个事实性问题，或一句待核查的论断。Ask a factual question, or give a claim to verify.
- **不适用 Not for** — 详尽的多源研究报告（→ deep-research）、主观 / 推荐类问题。Exhaustive research reports, or opinion / recommendation questions.
