串起整条 vince-skill 流水线（guidance → engineer → zipper），带质量门循环，把一个 skill 从想法做到工业级。
Drives the full vince-skill pipeline (guidance → engineer → zipper) end to end with quality-gate loops.

- **触发 Triggers** — “把这个 skill 从头建好并测试” · “run the whole skill pipeline” · `/vince-skill-conductor`
- **用法 Use** — 描述一个 skill 想法，或指向一个现有 skill。Describe a skill idea, or point at an existing skill.
- **不适用 Not for** — 只跑单个阶段（直接调那个阶段）、只做规划、只做压缩。A single stage only, planning-only, or compression-only.
