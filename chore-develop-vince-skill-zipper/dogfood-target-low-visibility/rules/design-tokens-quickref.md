# Design Token Quick Reference

Key values extracted from `references/design-tokens.yaml`.

## Touch & Sizing

- **Touch minimum**: 48x48dp Android, 44x44pt iOS
- **Primary button**: 64-80dp height
- **Option card**: 72-96dp height
- **Spacing**: 8dp minimum, 12dp preferred, 16dp for danger isolation

## Contrast

- **Text contrast**: ≥ 4.5:1 normal, ≥ 3:0 large text, ≥ 3:0 UI components

## Typography

- **Page title**: 22-28sp bold
- **Option label**: 20-26sp bold
- **Button label**: 18-22sp bold

## Themes & Scaling

- **Themes**: light, dark, high-contrast-light, high-contrast-dark, low-light-black
- **Text scale**: 70-400%
- **Button scale**: 50-200%
