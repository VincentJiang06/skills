# Research Mode

Load this file when the user asks about academic foundations, research,
citations, or the evidence base behind design decisions.

## Workflow

1. Load `references/research-bibliography.yaml` for the curated P0 reading
   list organized by research cluster:
   - Situational impairment
   - Haptic/audio feedback
   - Worker/low-literacy
   - Touch target
   - Accessibility standards

2. Map the user's question to the relevant research cluster and provide
   citations from the bibliography.

3. If the user needs deeper exploration, note the full reference details
   (DOI, authors, year) available in the YAML file for follow-up.
